Цель:
Научиться применять паттерн Presentation Model для реализации окон пользовательского интерфейса


Описание/Пошаговая инструкция выполнения домашнего задания:
Откройте папку Homeworks->PresentationModel. Там будут ассеты для дз
Сделать верстку попапа из UI нарезки (папка Sprites)
Сделайте фундамент для интерфейса (Canvas, Camera, EventSystem)
Напишите код для попапа и презентера
Проверьте работоспособность
Дополнительные задания (Опционально):
Напишите менеджер попапов
Разверните вашу архитектуру игры (DI, уход от монобехов)
Выложите домашнее задание в ветку homework_presentation_model

Критерии оценки:
Применение паттерна PresentationModel
